Electrode Boilerplate

Maximum Performance Out of the Box #

In our Getting Started section, we introduced you to our Yeoman Electrode Generator and we constructed an impressive application with the built in technologies that the simple yo electrode command gave us (read our Getting Started: What's Inside to learn more):

  • React

  • Redux

  • React Router

  • CSS Modules

  • Universal rendering

  • Webpack

  • Webpack Isomorphic Loader

  • Babel

  • ESLint

  • Mocha + Enzyme + TravisCI

  • Gulp

  • Yeoman

  • History

  • Bluebird

  • Electrode Confippet

  • Electrode-Server

  • Electrode-Docgen

This is the beginning foundation of the Electrode Boilerplate. You have learned in the Stand Alone Modules section that each of the Electrode modules are agnostic and can be used individually to enhance one specific area of your application. However, integrated together, you have a supercharged application; one that is capable of handling the complex problems we face at WalmartLabs and reach maximum efficiency and performance.

This boilerplate includes the Electrode Generator and it also has the following stand alone modules and tools built in for peak optimization out-of-the-box:

  • Above the Fold Rendering
  • Server Side Render Cache + Profiling
  • Stateless CSRF Validation
  • Electrode Redux Router Engine
  • Electrode Bundle Analyzer
  • Electrify

Start creating a more complex application today with the Electrode Boilerplate. We can't wait to see what you build!
